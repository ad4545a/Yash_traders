## ğŸš€ Getting Started

### Prerequisites
-   Flutter SDK installed.
-   Android Studio or VS Code.
-   Java 17 (recommended for Gradle).

### Installation

1.  **Clone the Repo**
    ```bash
    git clone https://github.com/ad4545a/Yash_traders.git
    cd Yash_Traders
    ```

2.  **Install Dependencies**
    ```bash
    flutter pub get
    ```

3.  **Setup Firebase**
    -   Add your `google-services.json` to `android/app/`.

4.  **Run the App**
    ```bash
    flutter run
    ```
